### 👨‍💻 About Me

I'm a QA Automation Engineer with **6.5+ years of experience** building test frameworks and CI/CD-integrated automation for **US Tax and FinTech** platforms. My focus is on designing scalable, maintainable test architectures — and, more recently, on applying AI and LLMs to make test creation and maintenance more autonomous.

- 🎭 Deep expertise in **Playwright** across UI, API, and end-to-end workflows
- 🤖 Building **AI-augmented testing** solutions — Playwright MCP, LLM-driven test generation, agentic QA workflows
- 🥒 Strong advocate for **BDD/Cucumber** to keep QA aligned with business requirements
- ☁️ Experienced with **CI/CD pipelines**, containerized test execution, and cloud environments
- 🌱 Currently exploring **AI agents for autonomous test maintenance**
- 💬 Happy to talk about test architecture, API automation, or AI in QA
